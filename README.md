📁 Web Portfolio - Maria Jose Chavarriaga

This is a basic web portfolio project built with HTML and CSS, showcasing personal information, projects, and an additional page dedicated to pets.

🧩 Project Structure

The project includes the following files:

📁 project/
│── index.html
│── mascotas.html
│── style.css
🌐 index.html

Main portfolio page. It contains:

Header with navigation menu
"About Me" section
Projects section using CSS Grid
Link to the pets page
Key Features:
Uses Flexbox for header layout
Internal navigation with anchor links
Grid-based layout for project cards
🐶 mascotas.html

Secondary page dedicated to displaying pet images.

Content:
Main title
Image gallery of dogs
Link to return to the portfolio
🎨 style.css

Main stylesheet for the project.

Includes:
🔹 Header with Flexbox
Horizontal alignment of elements
Space between title and navigation
🔹 Grid Layout for Projects
3-column layout
Consistent spacing between cards
🔹 Card Styling
Rounded borders
White background
Internal padding
🚀 Technologies Used
HTML5
CSS3
Flexbox
Grid Layout
⚠️ Important Notes
Make sure the CSS file name matches in both HTML files:
index.html uses: style.css
mascotas.html uses: styles.css ❗ (this may cause issues)

👉 Recommendation: use the same filename (style.css) in both files.
